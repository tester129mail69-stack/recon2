## Features

- **Subdomain Enumeration** ‚Äî Brute-force, passive DNS, certificate transparency
- **DNS Intelligence** ‚Äî All record types (A, AAAA, CNAME, MX, TXT, NS, SOA, SRV, CAA)
- **HTTP Probing** ‚Äî Status codes, response times, redirect chains, technology detection
- **Port Scanning** ‚Äî Fast async TCP port scanning with banner grabbing
- **Vulnerability Detection** ‚Äî CVE correlation, misconfiguration checks
- **Subdomain Takeover** ‚Äî Detect dangling DNS records pointing to unclaimed services
- **Cloud Asset Discovery** ‚Äî S3 buckets, Azure blobs, GCP storage, CloudFront
- **SSL/TLS Analysis** ‚Äî Certificate transparency, cipher suites, expiry dates
- **Email Security** ‚Äî SPF, DKIM, DMARC misconfiguration detection
- **Screenshots** ‚Äî Automated web screenshots for discovered assets
- **OSINT Integration** ‚Äî Shodan, Censys, VirusTotal, SecurityTrails, BinaryEdge
- **AI-Powered Analysis** ‚Äî Rule-based and OpenAI-backed risk scoring, executive summaries, remediation priorities
- **Multi-Target Campaigns** ‚Äî Scan multiple targets concurrently with aggregated reporting
- **Stealth Mode** ‚Äî Randomized delays, User-Agent rotation, proxy support
- **Webhook / SIEM** ‚Äî Push findings to Slack, Discord, generic webhooks, or Splunk HEC
- **SARIF Export** ‚Äî SARIF v2.1.0 output for enterprise CI/CD integration
- **Scan Checkpoints** ‚Äî Resume interrupted scans from where they left off
- **Adaptive Rate Limiter** ‚Äî Automatically backs off on 429/503 responses
- **Scope Configuration** ‚Äî YAML-based in-scope / out-of-scope rules (domains, CIDRs, wildcards)
- **Beautiful Reports** ‚Äî HTML, PDF, JSON, CSV, Markdown, SARIF output formats
- **Rich CLI** ‚Äî Colourised output with progress bars, tables, and ASCII art

## Configuration

Edit `config.yaml` to configure GODRECON:

```yaml
general:
  threads: 50        # Concurrent tasks
  timeout: 10        # Request timeout (seconds)
  retries: 3         # Retry attempts
  proxy: null        # Optional proxy URL
  output_dir: ./output

dns:
  resolvers:
    - "8.8.8.8"
    - "8.8.4.4"
    - "1.1.1.1"

modules:
  subdomains: true
  dns: true
  ports: false       # Disabled by default
  screenshots: false # Disabled by default

api_keys:
  shodan: ""
  virustotal: ""
```

Environment variable overrides follow the pattern `GODRECON__<SECTION>__<KEY>`:

```bash
export GODRECON__GENERAL__THREADS=100
export GODRECON__API_KEYS__SHODAN=your-key-here
```
